# SDK Examples

These examples show how to use a Payabli SDK in each supported language to build a small end-to-end sample app.

Each example is meant to demonstrate the same core flow:

1. Create a customer in your paypoint.
2. List existing customers.
3. Run a transaction flow that tokenizes a payment method with an embedded component and then submits the transaction through the SDK.
